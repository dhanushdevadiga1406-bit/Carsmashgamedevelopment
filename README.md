Car Smash is an exciting vehicle-based action and racing game developed to provide an engaging and immersive gaming experience. The game combines high-speed driving, collision mechanics, and vehicle destruction features, allowing players to compete in challenging environments while testing their driving skills.

The development of Car Smash focused on creating realistic vehicle physics, smooth gameplay mechanics, and visually appealing environments. The game includes multiple car models, obstacle-filled tracks, collision detection systems, score tracking, and progressive difficulty levels to keep players engaged.

  This is a code bundle for Car Smash Game Development. The original project is available at https://www.figma.com/design/WImB6L1ipx4Do1S1tm7Hde/Car-Smash-Game-Development.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
